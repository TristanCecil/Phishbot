業務がつながる

決算書を始めとした書類作成はもちろん、業務全体を楽にするのが freee の特徴。請求書の発行から売買掛金の管理、資金繰りの分析まで、お金周りの業務がこれ一つで完結します。経費精算など、会社全体で利用する機能も充実。

  1. 

    請求書・見積書

    請求書や見積書・納品書の作成が可能です。作成したデータは自動的に売掛金として登録されます。

  2. 売掛・買掛管理

    取引先別の売掛・買掛金の管理も freee で完結。期日を過ぎた取引があればお知らせします。

  3. 

    消込

    入出金データと売掛金買掛金のマッチングは freee が自動で推測。あとは確認するだけ。

  4. 

    各種レポート

    資金繰りレポート・売掛買掛レポートなど各種レポートを自動作成。リアルタイムに確認できます。

  5. 

    会計帳簿

    会社法に準拠した決算書や確定申告書が作成できます。損益計算書・貸借対照表・仕訳帳などにも対応。

安心のセキュリティ

freee は最新のセキュリティ技術を駆使し、お客様の大切な情報をお預かりしています。

lock全ての情報・通信を暗号化

お客様の個人情報・口座情報・入出金データなど、全てのデータを暗号化して保存。また全ての通信に金融機関レベルの 256bit SSL 通信方式を採用しています。

厳重なデータ保管

国内シェア No.1 の AWS を利用し、お客様のデータを厳重に管理。クラウド上でデータが保管されているので、パソコンの紛失など、物理的なデータ破損の心配はもうありません。

lock国際認証 TRUSTe 取得

freee は個人情報の扱いについて一定の基準を満たす企業が認定される国際的な認証 TRUSTe を取得しています。
