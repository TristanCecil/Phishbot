Airport delays

List shows airports with biggest traffic disruptions right now. The value between 0 and 5 is a balanced value that includes number of delayed flights, average delay and number of cancelled flights.

0 - 2 = Good traffic flow.

2 - 3.5 = Minor problems with some delays or few cancellations.

3.5 - 5 = Big problems with big delays and several flights cancelled.
